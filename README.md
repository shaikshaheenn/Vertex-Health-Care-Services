Vertex Healthcare Services is a full-stack healthcare web application built to provide healthcare-related information and services through a modern web interface and a robust backend system.

The project integrates frontend pages with a Node.js server, database connectivity, backend APIs, and email functionality.

---

## Technologies Used

### Frontend
- HTML5  
- CSS3  
- JavaScript  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB (MongoDB Atlas)

### Email Service
- Nodemailer

### Version Control
- Git  
- GitHub  

---

## Features

- Responsive healthcare website frontend
- Backend server using Node.js and Express
- RESTful APIs for handling application requests
- MongoDB integration for data storage
- Email notifications using Nodemailer
- Secure handling of sensitive data using environment variables
- Clean and deployment-ready project structure

---

## Project Overview

This project follows modern full-stack development practices.  
Frontend files handle the user interface, while backend APIs manage data processing, database operations, and email services. Sensitive configuration values are excluded from the repository for security purposes.

---

## Security

- Environment variables are used for database credentials and email configuration
- Sensitive files such as `.env` and `node_modules` are excluded from the repository
- SSL and HTTPS are handled at the hosting platform level

---

## Deployment Ready

- GitHub is used as the source repository
- Designed for deployment on cloud platforms such as Render
- Domain and SSL configuration are managed externally
